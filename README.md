# ReaScripts by themindbadger

This repository contains the following scripts:

- [**fx-lister**](https://github.com/mindbadger/mindbadger-reascripts/blob/master/fx-lister/README.md) : A UI tool for finding FX that match your needs while mixing, by quickly filtering on metadata that you define to describe your installed plugins.

The URL to import to ReaPack is https://raw.githubusercontent.com/mindbadger/mindbadger-reascripts/refs/heads/master/index.xml
